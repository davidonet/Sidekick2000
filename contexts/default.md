# MeetingScribe — Contexte par défaut

Ce fichier est chargé automatiquement quand aucun `--context` spécifique n'est fourni. Il fournit des consignes générales pour la génération de synthèses de réunions.

---

## Consignes de synthèse

- Être factuel : ne pas inventer d'information absente de la transcription.
- Identifier les intervenants par leur nom dès que possible.
- Quand un intervenant n'est pas identifiable, utiliser l'étiquette du transcript (SPEAKER_00, etc.).
- Regrouper les échanges par **thème** plutôt que par ordre strictement chronologique.
- Utiliser un ton professionnel et concis.
- Conserver les termes techniques tels qu'employés par les participants.
